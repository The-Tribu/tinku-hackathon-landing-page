# Tinku — Hackathon Norte del Valle del Cauca

Landing page for **Tinku**, the first hackathon in the Norte del Valle del Cauca region. A 24-hour event where developers and innovators come together to build real projects using AI and emerging technologies.

**April 17–18, 2026 · Cartago, Colombia · Universidad Cooperativa de Colombia**

## About

Tinku brings together 12 teams from 2 countries for 24 hours of non-stop hacking. Participants tackle challenges across five tracks: Sustainability, Artificial Intelligence, Education, Cloud AI, and Human & Agents.

Main prize: **$1,000 USD** + tech credits from Cursor, MiniMax, and Runway.

## Current Status

This is an early **design mockup** deployed as a static HTML page to validate the visual direction and content structure. The production site will be built with Astro, Svelte, TypeScript, and Tailwind CSS.

## Local Development

Open `index.html` in your browser — no build step required for this version.
